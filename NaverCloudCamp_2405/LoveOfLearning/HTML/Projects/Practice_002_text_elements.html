제목 <h1> ~ <h6> (heading) 위아래 여백 존재, big, bold 태그 표현하지 말기

단락 <p> (paragraph) 위아래 여백 존재

단락에서는 여러 번의 띄어쓰기와 줄 나누기는 오직 하나의 띄어쓰기로만 표현됨

단락에서 줄을
띄우고
싶으면
<br> (break line) 사용, 빈 태그(empty tag)임

            <pre> 태그(preformatted text)에서는 모든 띄어쓰기

            와 줄 나누기     가 모두 그대로 표현됨

가로선 <ht> (horizontal rule)

b(bold) 태그와 strong 태그는 강조표시, strong은 중요하다는 의미까지 내포

i(italic) 태그와 em(emphasized) 태그는 이탤릭체, em태그는 중요하다는 의미까지 내포

mark 태그는 highlighting 효과

del 태그는 삭제 효과

ins(insert) 태그는 밑줄, 즉 삽입 효과

위첨자는sup(superscript), 아래첨자는sub(subscript) 태그 사용

짧은 인용구는 q(quotation) 태그 사용

블록 인용구는

blockquote
태그 사용

용어의 축약형을 표현하려면 abbr 태그 사용

주소를 표현하려면

address 태그 사용, 이탤릭체와 위아래 공백 삽입됨

주석 사용은 <!--주석 내용--> 이렇게 사용, 여러 줄도 상관 없음. 단. 주석내에 주석 삽입은 금지!!!

엔티티(Entity)라고 HTML에서 사용하는 예약된 문자, 즉, 예약어가 있음

&엔티티이름; 혹은 &#엔티티숫자;

엔티티(entity)의 이름은 대소문자를 구분!!
